## Computer System Architecture - Labs ##
Repository for ITMO CSA Labs about CPU models and architectures

### Lab3
Simple scripts implementations for different CPU architectures

   - ```acc32``` - [sum_odd_n](acc32/sum_odd_n.asm)
   - ```f32a``` - [big_to_little_endian](f32a/big_to_little_endian.asm)
   - ```m68k``` - [stack_based_calculator](m68k/stack_based_calculator.asm)
   - ```risc-iv``` - [upper_case_cstr](risc-iv/upper_case_cstr.asm)
